WHERE 后面是查询条件，可以接and、or、>=、<=等。如果想查询满足多个条件的数据，可以这样做

注意：and、or等逻辑条件有优先级，为了保证查询条件的正确性，应该加上(),括号里面的条件总是最高优先级。

注意：WHERE 子句里面不能直接使用别名

正确的使用方式如下

自定义列名

DB2、Oracle、PostgreSQL→||。

MySQL→CONCAT

SQL Server→+

限定返回行数

DB2 使用FETCH FIRST子句。

MySQL 和 PostgreSQL 使用LIMIT子句。

Oracle 通过在WHERE子句中限制ROWNUM的值来获得指定行数的结果集。

注意：rownum的特殊性，永远取不到rownum = 5的数据

SQL Server 使用TOP关键字限定返回行数。
